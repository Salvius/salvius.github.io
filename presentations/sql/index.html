Union (∪)

The union of two sets A and B
is the set of elements which are
in A, in B, or in both A and B.

Example
A = {2, 4, 6, 8}
B = {1, 3, 5, 7}
A ∪ B = {1, 2, 3, 4, 5, 6, 7, 8}
