Open Source Robotics

Interested in learning more or possibly contributing?

All of the robot's source code is freely available for anyone to use. Salvius is a community driven development project which is rapidly becoming one of the most robust development platforms available. Many of the features provided through the robot's software can be used without purchasing any special hardware. Several apps are also available for Android, Amazon, and Chrome which can be downloaded and used to interact with the robot's API.

Salvius Travels

Salvius travels to a variety of locations. The goal of this initiative is to improve the project's development while simultaneously creating opportunities for people to work with new and cutting edge technologies. The robot has been the center of workshops and presentations, the robot also serves as a great platform for developing at hackathons and maker fairs.
